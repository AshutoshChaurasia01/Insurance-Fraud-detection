# Insurance-Fraud-detection
##Problem statement
Insurance fraud causes billions of dollars in losses annually.
This project builds a machine learning model to identify potentially fraudulent claims.
##Overview
This project predicts fraudulent insurance claims using machine learning.
The model analyzes customer, policy, and claim information to detect suspicious patterns.
##dataset
Dataset: Insurance Claims Fraud Detection (Kaggle)
Features: 39
Target: fraud_reported (Yes/No)
